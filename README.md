* Key OOP Concepts
Concept	Description
Class	Blueprint of objects
Object	Instance of a class
Constructor	Special method to initialize object
this keyword	Refers to current object
super keyword	Refers to parent class
static keyword	Belongs to class, not object
Advantages of OOP
Code Reusability
Better Maintainability
Data Security
Modularity
Easy Debugging

Real-Life Example
Car = Object
Engine, Wheels = Properties
Start(), Stop() = Methods

Conclusion

OOP helps developers build clean, efficient, and scalable applications by organizing code into logical structures.
